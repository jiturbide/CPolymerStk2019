Get the data from the json file using AJAX and display it in the content div
  1. Create an XMLHttpRequest object.
  2. Open an asynchronous GET request to the data/data.json URI.
  3. Add an onreadystatechange or onload callback to handle the response
  4. Inside the callback parse the JSON string contained inside the responseText property of the XMLHttpResponse object.
  5. Construct a String with the content and set it to the innerHTML of the content HTML element.
Look at the received object in the data.json file

Quick Reference

Create AJAX Request
  1. Create XMLHttpRequest Object.
    var xmlhttp = new XMLHttpRequest();
  2. Open the request with the HTTP method and url.
    xmlhttp.open(METHOD, URL, true);
    in this exercise the call would be:
    xmlhttp.open("GET", "data/data.json", true);
  3. Add the callback: 
    xmlhttp.onreadystatechange = function() {
  if (xmlhttp.readyState === 4 && xmlhttp.status === 200) {
    // handle response!
  }
};
    To parse a JSON response use:
    var personData = JSON.parse(xmlhttp.responseText);
  4. Send the request:
    xmlhttp.send();
DOM Manipulation
  1. Get the element
    var content = document.getElementById("content").innerHTML;
    Add/Remove child nodes
    content.appendChild(element);
    content.removeChild(element);
  2. Create elements using:
    document.createElement("TAG NAME");
    document.createTextNode("TEXT");
  3. Or simply use innerHTML:
    content.innerHTML = "SOME HTML";
