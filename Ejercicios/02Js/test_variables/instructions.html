In this exercise you have create functions using JavaScript.
  1. Split the data variable using var array = data.split(",") function.
  2. Assign person.firstName = array[0];
  3. Assign array[1] to person.lastName
  4. Parse array[2] to an integer and assign to person.age
  5. Parse array[6] to a float and assign to person.credit
  6. Trim the array[7] string and assign to person.rating
The person object should have the following values: 
person = {
  firstName:"John",
  lastName:"Doe",
  age:32,
  credit:153.25,
  rating:"A",
}
