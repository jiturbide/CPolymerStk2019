JavaScript and HTML5, Develop Web Applications

In this course you will learn how to develop Web Applications using JavaScript, HTML5 and CSS3.

Objectives

To build web applications by using JavaScript, HTML, and CSS, you would need to:

  • Use the Browser's tools to debug and inspect web applications and styles
  • Code application logic by using JavaScript to control user interactions and display data
  • Create applications with multiple HTML5 pages with navigation and HTML5 Forms to send data to services
  • Create design templates and standards by using CSS and JavaScript that adapt to different devices, including mobile, with Media Queries and Responsive Design
  • Read and validate data from HTML5 forms by using JavaScript
  • Use the JavaScript API to parse, modify, and validate data
  • Use JavaScript to add interactivity in HTML5 by using events and DOM modification
  • Use the JavaScript Object Notation to store and send JavaScript Object data to services, local storage, or across different pages and HTML5 elements
  • Draw on HTML5 canvas using JavaScript
  • Use the HTML5 Local Storage to store user data in web applications
  • Create JavaScript code to retrieve and display dynamic data from REST services by using AJAX
  • Create JavaScript code to interact with WebSockets for real-time communication.
